Common 2D Array For Loop Algorithms

Matrix Summation

Input: A 2D array of integers.

Process: Iterate through each row and column, adding all values to a running total.

Output: total sum of all numbers in the matrix

Row Sum

Input: A 2D array of integers.

Process: For each row, calculate the sum of its elements and store it in a new array.

Output: a 1D array where each element is the sum of a row

Column Sum

Input: A 2D array of integers.

Process: For each column, add up all values from top to bottom.

Output: a 1D array where each element is the sum of a column

Count Instances (Zeros)

Input: A 2D array of integers.

Process: Traverse all elements and count how many values are equal to zero.

Output: number of zero values in the matrix

Minimum Value

Input: A 2D array of integers.

Process: Compare each element to find the smallest value in the entire array.

Output: smallest number in the array

Maximum Value

Input: A 2D array of integers.

Process: Compare each element to find the largest value in the entire array.

Output: largest number in the array

Transformation (Double Values)

Input: A 2D array of integers.

Process: Create a new 2D array where each value is twice that of the original array.

Output: new 2D array of doubled values

Boolean Filter (Positive Map)

Input: A 2D array of integers.

Process: Create a new 2D array where each cell is true if the original value is positive and false otherwise.

Output: new 2D array of booleans

Diagonal Sum

Input: A square 2D array of integers.

Process: Add all elements where the row index equals the column index.

Output: sum of the main diagonal

Transpose Matrix

Input: A 2D array of integers.

Process: Create a new 2D array where rows become columns and columns become rows.

Output: the transposed matrix
